
# aircnc-react

Aircnc - Application to register spots on your office to developers book, based on their skills.

Made using React framework and NodeJS

**Back-End**

***install dependencies**: yarn install <br />
**run cmd**: yarn dev <br />
**port**: 3333*

**Front-End Web**

***install dependencies**: yarn install <br />
**run cmd**: yarn start <br />
**port**: 3000*
